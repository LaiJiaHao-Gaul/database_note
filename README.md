## 数据库基础语法

### use mongo

如果数据库不存在，则创建数据库，否则切换到指定数据库。

----------

### db.dropDatabase()

删除当前数据库

----------

### db

显示当前控制的数据库

----------

### showdbs

查看所有的数据库

----------

### db.mongo.insert( {"aaa":"bbb"} )

插入数据

----------

### use databaseA  switched to db databaseB
